# GUN Chat 

A decentralized chat app built with [GUN](https://gun.eco/) and svelte

- check out [GUN](https://gun.eco/) and [svelte](https://svelte.dev/) 
- thanks to [fireship.io's gun-chat tutorial](https://github.com/fireship-io/gun-chat)

## Setup

```
$ git clone <this-repo>
$ npm install
$ npm run dev
```
